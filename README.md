# 📚 Knowledge Hub

Knowledge Hub is a full-stack web application developed for Durham College, designed to centralize academic resources. Faculty can upload PDF materials, and students can browse and filter them dynamically through an intuitive interface.

## 👨‍💻 My Role
Team Leader • UI Development • Database Integration • Final Delivery

## 🚀 Features

### 📄 PDF Upload System
Faculty can upload PDF resources which are stored and managed in a MySQL database.

### 🔍 Dynamic Browse & Filter
Students can filter resources by category with real-time updates using AJAX (no page reload).

### 🎨 Durham College Branding
UI follows official Durham College branding guidelines including colors, fonts, and layout.

### ⚡ Improved User Experience
Interface optimized to reduce clicks and improve usability based on professor feedback.

---

## 🧠 Challenges & Solutions

### 📄 File Upload Issues
Resolved database connection and ensured proper PHP file handling for successful uploads.

### 🔄 AJAX Filtering Bug
Fixed issue by validating JSON responses and re-rendering UI using jQuery after each request.

### 🎨 Branding Consistency
Updated CSS based on official Durham College brand guidelines for accurate design implementation.
